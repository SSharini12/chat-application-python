# 💬 Chat Application using Python

## Description
A multi-user chat application built using Python socket programming that enables real-time communication between clients.

## Features
- Multiple users can connect
- Real-time messaging
- Username system
- Client-server architecture

## Technologies Used
- Python
- Socket Programming
- Threading

## How to Run
1. Run server.py
2. Run client.py
3. Enter server IP and username

## What I Learned
- TCP communication
- Multi-threading
- Client-server model
